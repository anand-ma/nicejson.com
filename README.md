# NiceJSON.com
## a web-based, privacy-focused JSON pretty printer

### Development

```
$ git clone git@github.com:crabl/nicejson.com.git
$ cd nicejson.com
$ npm install
$ npm run dev
```

To build the production artifacts:

```
npm run build
```
